# NYC Shooting Incident Analysis

This project explores youth involvement in NYC shooting incidents using statistical analysis and data visualisation in R.

## Overview
The analysis investigates the distribution of shooting perpetrators across different age groups and boroughs in New York City, with a particular focus on under-18 offenders.

The project also evaluates whether the proportion of under-18 perpetrators has increased beyond a previously reported statistic of 18% using a one-sample proportion hypothesis test.

## Tools Used
- R
- tidyverse
- ggplot2
- Quarto

## Features
- Data cleaning and preprocessing
- Exploratory data analysis
- Data visualisation using ggplot2
- Hypothesis testing
- Interpretation of real-world implications

## Key Findings
- The Bronx showed the highest concentration of under-18 shooting perpetrators.
- The hypothesis test did not provide strong statistical evidence that the proportion of under-18 perpetrators exceeded 18%.
- Despite this, youth involvement remains a significant concern and supports recommendations for targeted intervention programs.

## Dataset
NYPD Shooting Incident Data (NYC Open Data)
